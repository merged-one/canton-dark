# Canton Dark Phase 2 ATS RFQ Tests

This package contains script-based positive and negative workflows for directed multi-dealer
RFQ under `ATSPair`, including invite-set lock behavior, quote comparison ordering,
withdrawal and stale-quote rejection, response-window expiry, and blinded operator scope.

The local environment used for this prompt does not include the Daml SDK, so these scripts
were authored to match the production templates but were not executed here.
